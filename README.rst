===================
lcheapo
===================

Python tools for processing LCHEAPO data files

Overview
======================

Command-line programs:
----------------------

- lcfix: fix common bugs in an LCHEAPO file
- lcdump: dump raw information from LCHEAPO files
- lcheader: create an LCHEAPO header + directory
